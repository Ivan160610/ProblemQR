Про удалённую синхронизацию (быстрая инструкция)

Цель: иметь возможность вводить данные с телефона, чтобы они отображались в таблице на ПК.

1) Запустить сервер на ПК
- Откройте PowerShell или двойным щелчком запустите `run_server.bat`.
- Если используете PowerShell:

```powershell
cd "C:\Users\NastavnikiSklad\Desktop\Сайт"
.\run_server.ps1
```

- Если предпочитаете Windows-двойной клик, запустите `run_server.bat`.
- Скрипт попросит (опционально) authtoken ngrok, если вы хотите публичный доступ.

2) Локальный доступ по сети
- Узнайте IP ПК в сети:

```powershell
ipconfig
```

- На телефоне (в той же Wi‑Fi сети) откройте: `http://<IP_ПК>:3000`

3) Публичный доступ через ngrok
- Зарегистрируйтесь на https://ngrok.com и получите `authtoken` в личном кабинете.
- Выполните в PowerShell:

```powershell
ngrok authtoken <YOUR_TOKEN>
ngrok http 3000
```

- Скопируйте HTTPS адрес (например `https://abcd1234.ngrok.io`) и вставьте его в `Настройки → Сервер синхронизации` в приложении (поле "Сервер синхронизации"). Скрипт автоматически добавит `/api`.

4) Проверка синхронизации
- Добавьте запись на телефоне (войдите: логин `Bondarev`, пароль `147963`).
- На ПК обновите страницу — данные должны появиться.

5) Безопасность и рекомендации
- ngrok открывает доступ извне — не используйте в продакшене без аутентификации.
- Для публичного сервера замените простую авторизацию клиента на реальную (токены/серверную аутентификацию).

Если вы хотите, могу помочь шаг‑в‑шаг по телефону: пришлите вывод команды `npm start` и (или) URL ngrok, я проверю доступ и помогу настроить приложение.
